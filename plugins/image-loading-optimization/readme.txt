== Description ==

Improves accuracy of optimizing the loading of the LCP image by leveraging client-side detection with real user metrics. For more information, see the [overview issue](https://github.com/WordPress/performance/issues/869) on GitHub.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Image Loading Optimization**.
3. Install and activate the **Image Loading Optimization** plugin.

= Manual installation =

1. Upload the entire `image-loading-optimization` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the **Image Loading Optimization** plugin.
